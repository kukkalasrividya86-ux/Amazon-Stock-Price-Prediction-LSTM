# Amazon Stock Price Prediction using LSTM

## Project Overview
This project uses Long Short-Term Memory (LSTM) Neural Networks to predict Amazon stock prices based on historical stock market data.

## Features
- Data Preprocessing
- Stock Price Visualization
- LSTM Model Training
- Future Price Prediction
- Performance Evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- TensorFlow
- Keras
- Matplotlib
- Scikit-Learn

## Machine Learning Concepts
- Deep Learning
- Time Series Forecasting
- LSTM Networks
- Data Normalization

## Future Improvements
- Real-time stock prediction
- Hyperparameter tuning
- Multi-stock forecasting
